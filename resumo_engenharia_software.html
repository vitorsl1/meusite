Summary of Software Engineering

\n
\n
\n

Introduction

\n

Software engineering is the systematic application of engineering approaches to software development in a methodical way.

\n
\n
\n

Key Concepts

\n \n \n \n \n \n \n \n \n \n \n \n \n \n \n \n \n \n \n \n \n \n \n \n \n \n
ConceptDescription
Requirements AnalysisThe process of determining user expectations for a new or modified product.
DesignThe process of defining the architecture, components, and interfaces of a system.
ImplementationThe process of executing the design into a working software product.
TestingThe process of executing a program with the aim of finding errors.
MaintenanceThe process of modifying a software system after delivery to correct faults, improve performance, or adapt to a changed environment.
